# Resumen del Proyecto: Hub Interactivo de Circuitos Eléctricos II

Este proyecto es una plataforma educativa de alto rendimiento diseñada para facilitar el aprendizaje y la experimentación en la asignatura **Circuitos Eléctricos II (060-2413)** de la Universidad de Oriente. Utiliza una arquitectura moderna de Aplicación de Varias Páginas (MPA) con un enfoque en la interactividad y la visualización de datos en tiempo real.

## 🚀 Capacidades y Funcionalidades

### 1. Simuladores de Unidades (I - VI)
Cada unidad del programa académico cuenta con un laboratorio virtual específico:
- **Unidad I (RC/RL)**: Visualización de constantes de tiempo y respuestas transitorias.
- **Unidad II (RLC)**: Estudio de los estados de amortiguamiento (Sub, Sob, Crítico).
- **Unidad III (Espacio de Estados)**: Retrato de fase dinámico para variables de estado.
- **Unidad IV (Frecuencia)**: Generador de Diagramas de Bode (Magnitud y Fase).
- **Unidad V (Plano S)**: Análisis de estabilidad mediante la ubicación de polos en el plano complejo.
- **Unidad VI (Dos Puertos)**: Extracción matricial de parámetros Z, Y, h y ABCD.

### 2. Laboratorio RLC Master
Un simulador avanzado que incluye:
- **Niveles de dificultad**: Desde circuitos básicos hasta sistemas complejos con condiciones iniciales de energía preexistente (\(v_C(0)\) e \(i_L(0)\)).
- **Soluciones en tiempo real**: Comparativa visual entre la resolución en el dominio del tiempo y el dominio de Laplace.

### 3. Herramientas de Apoyo
- **Hoja de Fórmulas Inteligente**: Compendio interactivo de ecuaciones fundamentales organizado por unidades.
- **Conceptos y Analogías**: Sección teórica que utiliza analogías hidráulicas y mecánicas para construir intuición física sólida.
- **Programa Académico**: Acceso rápido a la ficha técnica, objetivos y bibliografía recomendada.

### 4. Sistema de Evaluación
- **Módulo de Autoevaluación**: Un test técnico de 6 preguntas (una por unidad) con retroalimentación inmediata, explicaciones matemáticas detalladas y generación de resultados gráficos.

## 🛠️ Stack Tecnológico

- **Frontend**: HTML5 Semántico y Tailwind CSS para un diseño *Glassmorphism* premium.
- **Motor Gráfico**: [Chart.js](https://www.chartjs.org/) para la representación de señales dinámicas y diagramas de fase.
- **Matemáticas**: [MathJax v3](https://www.mathjax.org/) para el renderizado impecable de ecuaciones en LaTeX.
- **Interactividad**: JavaScript Vanilla (ES6+) con lógica procedimental para simulación numérica (Métodos de integración como Euler para sistemas dinámicos).
- **Diseño UI**: Lucide Icons para una iconografía moderna y limpia.

## 🎨 Características de Diseño
- **Estética "Dark Mode" Premium**: Fondo pizarra profundo con degradados radiales dinámicos.
- **Responsividad Total**: Optimizado para dispositivos móviles, tablets y computadoras de escritorio.
- **Navegación Intuitiva**: Sistema de botones de retorno centralizado y dashboard de tarjetas interactivas.
