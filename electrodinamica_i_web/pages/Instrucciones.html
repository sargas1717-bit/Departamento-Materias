Guía de Usuario

APRENDE A UTILIZAR LOS SIMULADORES

SIMULADOR 3D

Explorador de Campos

1. Selecciona el tipo de campo (Radial, Rotacional, Gradiente).
2. Usa el ratón para rotar el espacio (Clic izquierdo del ratón).
3. Usa el scroll para hacer zoom y ver los vectores de cerca.

SIMULADOR 2D

Cargas Puntuales

1. Pulsa sobre una carga para moverla por el lienzo.
2. Añade nuevas cargas (+) o (-) para observar la superposición de campos.
3. La longitud de las flechas indica la intensidad relativa del campo eléctrico.

SIMULADOR B

Magnetostática

1. Ajusta la corriente (I) para ver cómo cambian los círculos de campo.
2. Si la corriente es negativa, la dirección del campo se invierte.
3. La vista lateral usa puntos (campo saliente) y cruces (campo entrante).
